# Sales-Performance-and-Distribution-Overview

# Project Impact
Developed an end-to-end sales analytics solution by designing a relational SQL database and building an interactive Power BI dashboard to analyze revenue and quantity performance across countries, channels, and warehouses.

# Business Objective

Enable stakeholders to monitor sales performance, identify seasonal trends, evaluate channel contribution, and support data-driven market and inventory decisions.

# Tools Used

SQL, Power BI

# Analysis

Designed a relational database with primary and foreign keys, cleaned and transformed data using SQL, and created an interactive Power BI dashboard for multi-dimensional sales analysis.

# Key Insights

Germany generated the highest revenue consistently across multiple years.

Bosnia, Belarus, and Moldova were among the lowest-performing markets.

Sales trends revealed seasonal revenue patterns across quarters.

Warehouse quantity analysis highlighted key distribution hubs for inventory movement.

# Business Value

Provided insights for market prioritization, channel optimization, and inventory planning.
